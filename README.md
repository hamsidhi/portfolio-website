# Hamza Siddiqui — Applied AI Engineer Portfolio

A premium, highly interactive personal portfolio built with modern web technologies. Designed to showcase applied AI systems, data analytics, and automation engineering with a product-led aesthetic.

## 🚀 Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion (3D Tilt Physics, Radial Tracking)
- **UI Components:** Shadcn UI + Skiper UI
- **Contact:** Native backend-free routing via FormSubmit

## ✨ Key Features
- **Dynamic Theming:** Seamless transitions between Dark (Premium default) and Beige (Reading mode) with zero hydration mismatch.
- **Antigravity Motion:** Custom 3D tilt tracking on project cards that actively follow the user's cursor.
- **Intelligent Cursor:** Global radial spotlight and cursor-trailing physics.
- **Staggered Page Transitions:** Global entry animations built directly into the Next.js routing template.

## 💻 Local Development
First, run the development server:
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
